All endpoints

Return JSON. CORS enabled. No authentication required. Click any card to try it.

GET/regions

All 17 PH regions with codes and regional centers.

?island_group=Luzon|Visayas|Mindanao
GET/holidays

PH holidays — regular and special non-working days.

?year=2026  ·  ?type=regular
GET/cities

Cities with population, province, and income class.

?region=R10  ·  ?province=Cebu
GET/schools

State and private universities, colleges, technical schools.

?region=R10  ·  ?type=State
GET/scholarshipsNEW

CHED, DOST-SEI, SM Foundation, Iskolar ng Bayan, and more.

?level=Undergraduate  ·  ?agency=DOST
GET/currencyNEW

Live PHP exchange rates + convert any amount.

/currency/convert?from=USD&to=PHP&amount=100
GET/weather/:cityNEW

Live weather for 20 PH cities (open-meteo).

/weather/manila  ·  /weather/cebu-city

For students 🎓

How you can use this API in your school work.

📊

Capstone projects

Build a thesis/capstone app using real PH data. Tourism app? Use /regions + /weather. Enrollment system? Use /schools. Scholarship finder? Use /scholarships.

📝

Research papers

Cite live data in your thesis. PH demographics, holidays, university lists — all in JSON, easy to import to Excel/SPSS/Python.

🤖

Learn to code

Practice with a real API. Build a fetch() exercise, learn JSON parsing, try in HTML/JS, Python requests, or React.

📚

Homework help

Kuya AI explains coding, math, Filipino, and study skills in language you'll actually understand.

💰

Scholarship search

Find financial aid you might qualify for — CHED, DOST-SEI, SM Foundation, Iskolar ng Bayan, and more.

🌦️

School cancellations

Use /weather/:city + /holidays to build an "is there class today?" widget. Or a typhoon-aware calendar.

Code examples

Copy a snippet, paste it into your project.

cURL
curl https://ph-public-api.vercel.app/scholarships
JavaScript (fetch)
const res = await fetch(
  'https://ph-public-api.vercel.app/weather/cebu-city'
);
const data = await res.json();
console.log(data.current.temperature_2m);
Python (requests)
import requests
r = requests.get(
  "https://ph-public-api.vercel.app/currency/convert",
  params={"from":"USD","to":"PHP","amount":100}
)
print(r.json())
Ask Kuya AI (POST)
fetch('https://ph-public-api.vercel.app/ai', {
  method: 'POST',
  headers: {'Content-Type':'application/json'},
  body: JSON.stringify({
    question: 'Paano mag-fetch ng API sa JavaScript?'
  })
}).then(r => r.json()).then(console.log);
