Endpoints

All endpoints return JSON. CORS enabled. No authentication required.

GET/regions

All 17 regions with codes, island group, and regional center.

?island_group=Luzon|Visayas|Mindanao
GET/regions/:code

Single region by code: NCR, CAR, R1–R13, BARMM.

example: /regions/R10
GET/holidays

PH holidays — regular and special non-working.

?year=2026  ·  ?type=regular
GET/cities

Cities with population, province, and income class.

?region=R10  ·  ?province=Cebu
GET/schools

State and private universities, colleges, technical schools.

?region=R10  ·  ?type=State
GET/schools/:code

A specific school by code (UP, MOSCAT, USTP, etc).

example: /schools/MOSCAT

Examples

Copy a snippet, paste it into your project.

cURL
curl https://ph-public-api.vercel.app/regions?island_group=Mindanao
JavaScript (fetch)
const res = await fetch(
  'https://ph-public-api.vercel.app/holidays?year=2026'
);
const { data } = await res.json();
Python (requests)
import requests
r = requests.get(
  "https://ph-public-api.vercel.app/schools",
  params={"region": "R10"}
)
data = r.json()["data"]
